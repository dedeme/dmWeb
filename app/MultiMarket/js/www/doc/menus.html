Main

start()

  • url[0] is null => Connection
    • Connected? => run()
    • else => Authentication()
  • else => run()

run()

Check url[0]:

  • Is null? => Show Sys
  • Is "market"? => Show Market
  • Is "daily"? => Show Daily
  • Is "fleas"? => Show Fleas
  • else => Message of error => run with url[0] = null

C client-server applications

1. Copy Dummy directory

Example:

We copy Dummy as MyMarket

2. In main directory

  • Remove doc directory
  • Change name of Dummy.geany (e.g. MyMarket.geany)
  • Run geany file and:
    • Change page name in Construir -> Establecer commandos de construcciÃ³n -> Browser
    • Change Project / Properties

3. In server

  • Remove file c/bin/Dummy
  • Update dependencies.
  • Change Make file. Al least "PRG=Dummy".
  • In main.c change app_name, data_version, app_dir, and expiration.
  • Compile server.

4. In client

  • Delete symbolic lynk js/www/tmp
  • Empty directory js/www/js
  • Change title in js/www/index-clean.html and js/www/index-closure.html
  • Update dependencies.
  • Change application name and version in js/src/Main.js

5. In www/dmcgi

Make a symbolic link to js/www with application name (e.g. MyMarket -> client/www)

6. In wwwcgi

  • Make a symbolic link in wwwcgi/dmcgi/bin to executable
    (e.g. MyMarket -> c/bin/MyMarket)
  • Set a link in wwwcgi/dmcgi/ccgi.sh

7. Steps end

  • In editor run Compile All
  • Visit localhost page launching Browser.
  • If application uses a tmp directory, make a symbolic link to wwwcgi/dmcgi/APP_NAME/tmp. For example, from MiMarket/www write:
    ln -s /dm/wwwcgi/dmcgi/MyMarket/tmp tmp

Nicks table.

Module signature

{
add data del getModel getNick init nicks selectedNicks setIsSel setModel setName
add lk nickName is sync.t → string → {nickId is number, ok is boolean},
Adds a new nick if it is not duplicated. In such case it logs an error and returns "false". If the operation succeeds, the nick identifier is returned.
lk-Synchronization lock.
nickName-Name of nick to add.
data lk is sync.t → {lst is list<nick.t>, model is number},
Returns list and model of nicks.
lk-Synchronization lock.
del lk nickId is sync.t → number → (),
Removes the nick with id "nickId".
lk-Synchronization lock.
nickId-Identifier of nick to delete.
getModel lk is sync.t → (None () | Some nick.t),
Returns Nick model. If it is not defined, returns the any nick. If an error happens, returns none.
lk-Synchronization lock.
getNick lk nickId is sync.t → number → (None () | Some nick.t),
Returns the nick which id is 'id'.
lk-Synchronization lock.
nickId-Nick to search.
init parent is string → (),
Initializes configuration table.
parent-Parent directory.
nicks lk is sync.t → list<nick.t>,
Returns the list of nicks.
lk-Synchronization lock.
selectedNicks lk is sync.t → list<nick.t>,
Returns the list of selected nicks.
lk-Synchronization lock.
setIsSel lk nickId value is sync.t → number → boolean → (),
Selects/Deselects the nick with id "nickId"
lk-Synchronization lock.
nickId-Identifier of nick to select/deselect.
value-'true' If selected.
setModel lk nickId is sync.t → number → (),
Set the nick with id "nickId" as nick model.
lk-Synchronization lock.
nickId-Identifier of nick to select/deselect.
setName lk nickId name is sync.t → number → string → (None () | Some string)
Modifies the name of nick with id "nickId".

If the name is duplicated or nickId does no exist, it returns 'false'.
lk-Synchronization lock.
nickId-Identifier of nick to modify.
name-New name.
}

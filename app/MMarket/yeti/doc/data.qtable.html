module data.qtable

Table with data of nicks and quotes.

typedef t = {
nicks is array<string>,
Nick names.
values is double[][]
Quote values (closes, opens...) of nicks.

Description

They are "double[rows][colums]". 
 - rows are 'days' dates.
 - colums are 'length nicks' companies.
}
